    斜率优化
    2022-08-24|Study

    斜率优化可以在动态规划中根据斜率的性质优化时间。

    当转移出现ai×bja_i\times b_jai​×bj​的时候，单调队列优化不再适用，就要使用斜率优化。

    例题

    [HNOI2008]玩具装箱
    给定nnn个玩具，每个玩具价值CiC_iCi​，分成若干段，定义分段代价cost(i,j)=(j−i+∑k=ijCj−L)2,i≤jcost(i,j)=(j-i+\sum_{k=i}^jC_j -L)^2,i\leq jcost(i,j)=(j−i+∑k=ij​Cj​−L)2,i≤j,LLL为给定常数,求分段的最小代价和。
    1≤n≤5×104,1≤L,Ci≤1071\leq n \leq 5\times 10^4,1\leq L,C_i\leq 10^71≤n≤5×104,1≤L,Ci​≤107

    朴素做法

    fif_ifi​表示前iii个物品分段的最小代价,preipre_iprei​为价值前缀和。

    fi=min⁡j<i{fj+(i−j−1+prei−prej−L)2}f_i=\min_{j<i}\{f_j+(i-j-1+pre_i-pre_j-L)^2\} fi​=j<imin​{fj​+(i−j−1+prei​−prej​−L)2}

    时间复杂度O(n2)O(n^2)O(n2)，无法通过本题。

    优化

    设si=i+preis_i=i+pre_isi​=i+prei​,L=L+1L=L+1L=L+1

    fi=min⁡j<i{fj+(si−sj−L)2}=min⁡j<i{fj+(si−L)2+2(L−si)sj+sj2}f_i=\min_{j<i}\{f_j+(s_i-s_j-L)^2\}=\min_{j<i}\{f_j+(s_i-L)^2+2(L-s_i)s_j+s_j^2\} fi​=j<imin​{fj​+(si​−sj​−L)2}=j<imin​{fj​+(si​−L)2+2(L−si​)sj​+sj2​}

    fi−(si−L)2=min⁡j<i{fj+sj2+2sj(L−si)}f_i-(s_i-L)^2=\min_{j<i}\{f_j+s_j^2+2s_j(L-s_i)\} fi​−(si​−L)2=j<imin​{fj​+sj2​+2sj​(L−si​)}

    将fi−(si−L)2f_i-(s_i-L)^2fi​−(si​−L)2看作bib_ibi​，fj+sj2f_j+s_j^2fj​+sj2​看作yjy_jyj​，sjs_jsj​看作xjx_jxj​，−2(L−si)-2(L-s_i)−2(L−si​)看作kik_iki​，就有

    bi=min⁡j<i{yj−kixj}b_i=\min_{j<i}\{y_j-k_ix_j\} bi​=j<imin​{yj​−ki​xj​}

    将(xj,yj)(x_j,y_j)(xj​,yj​)看作坐标系的点，于是转换成定斜率过一点求最小截距。

    几个性质：

    1. 最优决策点在下凸壳，可以直接维护凸包。
    2. 本题的斜率单调递增。
    3. 决策单调性：cost(i,j)=(si−sj−L)2cost(i,j)=(s_i-s_j-L)^2cost(i,j)=(si​−sj​−L)2是四边形不等式(同时斜率单增也可证明)。

    所以可以用单调队列维护一个凸包，具体做法如下：

    1. 弹队首，条件为队首代表线段斜率小于 kik_iki​，此时选下一个点肯定更优。

    2. 队首为最优决策点，更新(决策单调性)。

    3. 加入(xi,yi)(x_i,y_i)(xi​,yi​)决策点，出队当且仅当新的凸包不包含队尾节点(比较斜率)。

    4. 加入当前决策点。

    复杂度O(n)O(n)O(n)，完美解决。

    进阶

    如果斜率不单调怎么办？即决策也不单调。

    但这仍然满足最优决策点在凸包上，且新增决策点x单增，我们仍然维护凸包上的点，对于给定斜率kkk，最优决策点一定是在凸包上的直线刚恰好切上凸包的点，满足单调性，故可以用二分答案去找最优决策点。

    [SDOI2012]任务安排
    将nnn个任务分段并顺序完成，给定完成时间TTT和费用系数CCC，完成一段前需冷却sss，完成时间为这段时间之和，每个任务的价值为完成时间×\times×费用系数，最小化价值和。
    1≤n≤3×105,1≤s≤28,∣Ti∣≤28,0≤Ci≤281 \leq n \leq 3 \times 10^5,1\leq s \leq 2^8,|T_i|\leq 2^8,0 \leq C_i\leq 2^81≤n≤3×105,1≤s≤28,∣Ti​∣≤28,0≤Ci​≤28

    先看O(n2)O(n^2)O(n2)的做法，fif_ifi​表示前iii个任务的最小价值和。

    最朴素做法中需要一维分了几组的状态，本题不要求分组数量，故有所多余。

    这里有个trick：转移的时候，根据乘法分配率，对[j,i][j,i][j,i]分组，iii以后的任务都多了一个s×Cis\times C_is×Ci​的价值，可以直接加上去,于是转移就分为准备时间的贡献和做任务的贡献,这时转移的时候就不必计算当前批任务的具体完成时间，改为计算不计冷却的完成时间。

    设时间前缀ptipt_ipti​,价值前缀pcipc_ipci​

    fi=min⁡j<i{fj+pti×(pci−pcj)+s×(pcn−pci)}f_i=\min_{j < i}\{f_j+pt_i\times(pc_i-pc_j)+s\times(pc_n-pc_i)\} fi​=j<imin​{fj​+pti​×(pci​−pcj​)+s×(pcn​−pci​)}

    优化

    fi=min⁡j<i{fj+ptipci−ptipcj+pcns−pcis}f_i=\min_{j<i}\{f_j+pt_ipc_i-pt_ipc_j+pc_ns-pc_is\} fi​=j<imin​{fj​+pti​pci​−pti​pcj​+pcn​s−pci​s}

    fi−ptipci−pcns+pcis=min⁡j<i{fj−ptipcj}f_i-pt_ipc_i-pc_ns+pc_is=\min_{j<i}\{f_j-pt_ipc_j\} fi​−pti​pci​−pcn​s+pci​s=j<imin​{fj​−pti​pcj​}

    bi=fi−ptipci−pcns+pcisb_i=f_i-pt_ipc_i-pc_ns+pc_is bi​=fi​−pti​pci​−pcn​s+pci​s

    yj=fj,xj=pcj,ki=ptiy_j=f_j,x_j=pc_j,k_i=pt_i yj​=fj​,xj​=pcj​,ki​=pti​

    bi=min⁡j<i{yj−kixj}b_i=\min_{j<i}\{y_j-k_ix_j\} bi​=j<imin​{yj​−ki​xj​}

    这里由于TiT_iTi​可负，斜率不再单调，但xxx仍然单调，利用上文做法，运用二分答案找最优决策点。

    复杂度O(nlogn)O(nlogn)O(nlogn)

    再进阶

    如果xxx也不单调怎么办，维护凸包就不能用队列维护，但我们有平衡树！但是实现比较繁琐。(代码能力太弱)

    以玩具装箱题目为例子，这里∣Ci∣≤107|C_i|\leq 10^7∣Ci​∣≤107。

    我们发现只需要使x,kx,kx,k单调就能用基础做法解决问题，可以使用创造单调性的利器CDQ分治创造单调性，具体的：例如分治[l,r][l,r][l,r]，加入决策点[l,mid][l,mid][l,mid]，更新状态[mid+1,r][mid+1,r][mid+1,r]，右侧没有考虑到的决策点会在向右递归的时候处理到，且[l,mid][l,mid][l,mid]的状态的答案一定已经处理完全，所以正确性得证，复杂度O(nlog2n)O(nlog^2n)O(nlog2n)，多出的logloglog在于左区间xxx的排序和右区间kkk的排序。
